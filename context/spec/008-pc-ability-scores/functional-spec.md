# Functional Specification: Player Character Ability Scores

- **Roadmap Item:** Player Character ability score generation within the Character table set
- **Status:** Completed
- **Author:** Claude

---

## 1. Overview and Rationale (The "Why")

When creating a Player Character in Vaults of Vaarn, a GM rolls for six ability scores: **Strength, Dexterity, Constitution, Intellect, Psyche, and Ego**. Each ability is determined by rolling 3d6 and taking the **lowest die** as that ability's bonus. Currently, the Player Character table set only generates an Ancestry (which triggers ancestry-specific sub-tables). The GM still has to roll physical dice for ability scores separately, which breaks the flow of one-click character generation.

By including ability scores in the Player Character roll, the tool delivers a more complete character generation experience in a single click — Ancestry plus all six ability bonuses — reducing the need to reach for physical dice during prep or play.

**Success Criteria:** When a GM rolls on the Player Character table set, the result includes all six ability scores alongside Ancestry and ancestry-specific traits, making physical dice unnecessary for character creation.

---

## 2. Functional Requirements (The "What")

- **As a** GM, **I want** ability scores to be generated automatically when I roll on the Player Character table set, **so that** I get a complete character in one click.
  - **Acceptance Criteria:**
    - [x] Rolling the Player Character table set generates results for all six abilities: Strength, Dexterity, Constitution, Intellect, Psyche, and Ego — in that order.
    - [x] Each ability score is generated by rolling 3d6 and taking the **lowest** of the three dice as the bonus.
    - [x] The bonus value is displayed as the result for each ability (e.g., "Strength: 2"). The individual dice are not shown.
    - [x] The ability scores appear alongside the Ancestry result and any triggered ancestry sub-table results.
    - [x] Re-rolling the Player Character table set regenerates all six ability scores together (individual re-roll is not supported).
    - [x] Ability score results are captured in the session log and can be saved, just like any other generated result.

---

## 3. Scope and Boundaries

### In-Scope

- Generating six ability scores (Strength, Dexterity, Constitution, Intellect, Psyche, Ego) as part of the Player Character roll.
- "3d6 take lowest" dice mechanic for computing each ability bonus.
- Displaying the bonus value only.

### Out-of-Scope

- **Ability swap rules** — the rules allow swapping two ability scores after rolling; this is not included.
- **Individual ability re-rolls** — re-rolling regenerates all six scores together.
- **Showing individual dice** — only the derived bonus is displayed.
- **Hit Points or other derived stats** — only the six core abilities are generated.
- All other roadmap items (Session Log, Save Individual Results, Responsive Layout, Favorites/Pinned Tables) are separate features.
