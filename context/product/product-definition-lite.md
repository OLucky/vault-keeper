# Vault Keeper — Product Summary

## Vision

A fast, browser-based random generator tool for Vaults of Vaarn gamemasters that replaces flipping through sourcebooks with instant, one-click table rolls during live sessions.

## Target Audience

Personal use — a single GM running Vaults of Vaarn sessions who needs quick random generation at the table.

## Core Features

- **Category-based table browsing** — tables organized by type (NPCs, Weapons/Items), with more categories added over time.
- **One-click rolling** — generate a full result from a multi-table set with a single action.
- **Session log** — running log of generated results, exportable as plain text.
- **JSON-driven data** — all table content defined in structured JSON files for easy extensibility.
- **Sample data included** — ships with placeholder content; real Vaarn tables added over time.
